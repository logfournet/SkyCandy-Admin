# Về Config

Danh mục này sẽ hướng dẫn tổng quan về cấu trúc config của plugin SkyCandyPack

Config này sẽ nằm tại "plugins/SkyCandyPack" tại Panel

![image_8.png](image_8.png)

* **assets**: Chứa tên và thông tin của gói ResourcePack (Không nên đụng vào vì thường đã thiết kế sẵn)
* **build**: Chứa tệp tin đã xây dựng thành công cùng với file zip (Không nên đụng vào vì không quan trọng)
* **pack**: Nơi thêm model item, entity, customsound (Tùy chỉnh pack tại đây)
* **config.yml**: Config mặc định của SkyCandyPack (Không nên đụng vì đã config sẵn rồi)

**Trong Mục Pack Sẽ Có**

![image_9.png](image_9.png)

* **font**: Tùy chỉnh **Custom Font Image**. Có thể thêm hình ảnh hay icon vào Minecraft (Bạn có 1 kí tự nào đó mà muốn khi ghi kí tự đó ra sẽ hiện 1 hình ảnh. Kích thước hình ảnh có thể config)
![image_11.png](image_11.png)

* **item**: Thêm model custom của các vật phẩm, giáp, vũ khí...
![image_12.png](image_12.png)

* **model**: Thêm model entity mới vào trong game
![image_13.png](image_13.png)

* **sound**: Thêm âm thanh tùy chỉnh vào trong game
![image_14.png](image_14.png)
