# Lệnh Cơ Bản

| Lệnh                | Thông Tin                          |
|---------------------|------------------------------------|
| /skycandypack build | Xây dựng nhanh gói ResourcePack    |

| /skycandypack font   | Xem danh sách custom font image    | 
| /skycandypack debug   | Danh sách người chơi đang cài pack |
